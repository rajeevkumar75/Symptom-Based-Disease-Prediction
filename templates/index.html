    How this prototype works

    1. The model was trained on a curated symptom–disease dataset.
    2. Each symptom maps to a feature column (0/1) in that dataset.
    3. Your selected symptoms are converted into that feature vector and fed to the model.
    4. The predicted label is then decoded back into a disease name.

    This interface is provided strictly for educational and experimental purposes. It is not a substitute for professional medical advice, diagnosis, or treatment.
